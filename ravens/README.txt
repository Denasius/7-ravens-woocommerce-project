Кнопка "Перейти" на слайдере находится:
	Настрйки-> Общие -> В самом низу настроек

Блок с социальными сетями:
	Внешний вид -> Виджеты -> виджет "Социальные сети". 
Для создания ссылок на социальные сети нужно прописать в теге <a> вместо символа "#" ссылку на необходимую соц сеть. При желании поменять иконки соц.сетей вместо атрибута "fa-instagram(или др.)", нужно указать необходимый атрибут. Примеры можно посмотреть на сайте https://fontawesome.com/v4.7.0/ в разделе "Icons"
